# Interactive Setup Wizard

The easiest way to get started with Muti Metroo is using the interactive setup wizard.

## Running the Wizard

```bash
./build/muti-metroo setup
```

The wizard guides you through all configuration steps and can automatically:
- Generate TLS certificates
- Create agent identity
- Configure agent roles (ingress, transit, exit)
- Set up peer connections
- Install as system service (Linux/Windows)
